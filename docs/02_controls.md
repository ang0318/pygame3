# 操作键位

## 游戏中

| 键位 | 功能 |
|------|------|
| `A` / `←` | 向左移动 |
| `D` / `→` | 向右移动 |
| `Space` / `W` / `↑` | 跳跃 |
| `E` | 与最近 NPC 对话（靠近后出现提示） |
| `ESC` | 返回主菜单 |

### 对话框中

| 键位 | 功能 |
|------|------|
| `↑` / `↓` | 切换选项 |
| `Enter` | 确认 / 跳过打字机动画 |
| `ESC` | 中断对话（保留进度） |
| `C` | **开发者跳过**：强制当前问答判为正确 |

## 主菜单

| 键位 | 功能 |
|------|------|
| `↑` / `W` | 上移光标 |
| `↓` / `S` | 下移光标 |
| `Enter` / `Space` | 确认选项 |

## 音乐控制（任意场景）

| 键位 | 功能 |
|------|------|
| `M` | 切换静音 / 恢复 |
| `,`（逗号）| 降低音量 10% |
| `.`（句号）| 提高音量 10% |
| 点击右上角控件 | 切换静音 |

## 通关页

| 键位 | 功能 |
|------|------|
| `Enter` / `Space` | 返回主菜单 |
| `ESC` | 退出游戏 |
