## HUD — 顶部状态栏

文件：[`ui/hud.py`](../ui/hud.py)

顶部半透明黑条，显示分数（左）、关卡名（中）、生命（右）。

### 创建 & 使用

```python
hud = HUD(settings, font)

hud.score      = 0        # 当前分数
hud.lives      = 3        # 当前生命数
hud.level_name = "关卡 1" # 顶部中央文字

# 在 draw() 中调用
hud.draw(screen)
```

### 分数更新

`BaseLevelScene` 内部已集成：
- 答对问题：`hud.score += 100`
- 捡到宝石：`hud.score += 50`

你可以在子类任意位置修改 `self.hud.score`。

## DialogueBox — 对话框

文件：[`ui/dialogue_box.py`](../ui/dialogue_box.py)

底部弹出的对话框，支持打字机效果和多选项。

### 状态机

```
close ──open()──> active（打字机播放中）
                       ↓ 打字完成
               等待玩家操作
               ├── 无选项：Enter → confirmed=True
               └── 有选项：↑↓选择 → Enter → confirmed=True
                              ↓
                    show_feedback() → 显示正确/错误动画
                              ↓ 动画结束
                    feedback_done=True  （外部消费）
```

### API

```python
box = DialogueBox(settings, font, small_font)

# 打开对话框
box.open("文本内容", choices=["选项A", "选项B", "选项C"])
box.open("普通文字")   # choices=None 时只有「继续」提示

# 每帧
box.update(dt)
box.handle_event(event)
box.draw(screen)

# 读取状态
box.active          # 是否激活
box.confirmed       # 玩家按下 Enter（消费后需置 False）
box.selected_index  # 当前选中选项的索引
box.feedback_done   # 反馈动画结束（消费后需置 False）

# 显示答题反馈（1.2 秒绿/红闪烁）
box.show_feedback(correct=True)

# 关闭
box.close()
```

### 文本换行

支持 `\n` 强制换行，也支持中英文混排自动换行：

```python
box.open("第一行\n第二行，自动换行长文本会在超出宽度后折行。")
```

### 打字机速度

默认 40 字符/秒，可在源码中修改 `_char_speed`：

```python
self._char_speed = 60.0   # 更快
```

## MusicControl — 音乐控制条

文件：[`ui/music_control.py`](../ui/music_control.py)

右上角迷你控件（130×28px），**纯代码绘制，不依赖任何图片**。浮于所有场景最顶层，由 `main.py` 统一管理。

### 功能

| 操作 | 效果 |
|------|------|
| `M` 键 | 切换静音 |
| `,` 键 | 音量 -10% |
| `.` 键 | 音量 +10% |
| 点击控件 | 切换静音 |

### 在 main.py 中的使用方式

```python
music_ui = MusicControl(cfg, assets)

# 事件循环中
music_ui.handle_event(event)

# 绘制（在 display.flip() 前，必须在所有 mgr.draw() 之后）
music_ui.draw(screen)
```

> 音乐控制条挂在 `main.py` 而非场景中，这样场景切换时它不会消失。
