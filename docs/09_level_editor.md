# 关卡编辑器使用手册

---

## 启动

```bash
python edit.py
```

终端会列出所有 `levels/*_layout.json`，输入编号后进入 pygame 编辑窗口。

---

## 界面说明

```
┌─────────────────────────────────────────────────────┐
│  工具栏（顶部 40px）：关卡名 | 视角偏移 | 物体数 | 状态  │
├─────────────────────────────────────────────────────┤
│                                                     │
│   编辑区域（网格背景）                                │
│                                                     │
│   红色竖线 = 世界右边界                              │
└─────────────────────────────────────────────────────┘
```

---

## 键盘操作

| 键 | 功能 |
|----|------|
| `←` / `A` | 视图左移 |
| `→` / `D` | 视图右移 |
| `P` | 在视图中央添加平台 |
| `N` | 在视图中央添加 NPC |
| `G` | 在视图中央添加宝石 |
| `H` | 切换选中物体的**隐藏状态** |
| `Del` | 删除选中物体（出生点不可删） |
| `Ctrl+Z` | 撤回（最多 50 步） |
| `S` | 保存（覆写 JSON） |
| `ESC` | 退出编辑器 |

---

## 鼠标操作

| 操作 | 功能 |
|------|------|
| 左键点击 | 选中物体 |
| 左键拖拽 | 移动物体 |
| 右键点击 | 删除物体（出生点不可删） |
| 拖拽平台右下角红点 | 调整平台宽高 |
| 滚轮上下 | 横向滚动视图 |

## 物体类型

### 出生点（绿色轮廓矩形）
- 每个关卡只有一个，不可删除
- 拖动可调整玩家出生位置

### 平台
- 拖动：改变位置
- 拖拽右下角红色把手：改变宽高
- `H` 键：切换隐形（游戏中透明但保留碰撞）

### NPC（橙色人形）
- 编辑器只管位置，对话内容在 `scenes/levelN.py` 的 `_DIALOGUES` 中定义
- NPC 的 `dialogue_key` 需与代码中的字典 key 对应
- `H` 键：隐藏 NPC（游戏中不创建，不可交互）

### 宝石（黄色菱形）
- 仅关卡 2 使用，其他关卡也可添加（需在场景代码中读取）
- `H` 键：隐藏宝石（游戏中不生成）

---

## 隐藏状态说明

| 物体类型 | `hidden=true` 效果 |
|----------|-------------------|
| 平台 | 游戏中**不可见**，但碰撞完全有效（适合配合背景图使用） |
| NPC | 游戏中**不创建**，不可见不可交互 |
| 宝石 | 游戏中**不生成** |

编辑器中隐藏的物体会以深灰半透明显示，便于识别。

## JSON 格式说明

保存后的 JSON 结构（`levels/levelN_layout.json`）：

```json
{
  "meta": {
    "name": "关卡名称",
    "world_w": 1600
  },
  "player_spawn": { "x": 80, "y": 460 },
  "platforms": [
    { "x": 0, "y": 500, "w": 1600, "h": 40, "sprite_key": "default", "hidden": true },
    { "x": 300, "y": 400, "w": 200, "h": 20, "sprite_key": "default" }
  ],
  "npcs": [
    {
      "x": 400, "y": 400,
      "name": "向导",
      "sprite_key": "guide",
      "dialogue_key": "guide"
    }
  ],
  "gems": [
    { "x": 500, "y": 300 }
  ]
}
```

### meta 字段

| 字段 | 说明 |
|------|------|
| `name` | 关卡显示名（HUD 顶部） |
| `world_w` | 关卡总宽度（px），决定摄像机范围 |

---

## 常见工作流

1. `python edit.py` → 选择关卡
2. `A/D` 滚动视图，找到要修改的区域
3. 左键选中物体，拖动调整位置
4. 平台右下角红点调整尺寸
5. `H` 切换隐形地面（配合背景图）
6. `P/N/G` 添加新物体
7. `Ctrl+Z` 撤回误操作
8. `S` 保存，`python main.py` 验证效果
